# 链尚ant-design-vue后台管理系统的开发组件框架模版

> 基于vue,vuex,vue-router,ant-design-vue,axios的开发组件框架模版

## 用法

``` bash
$ npm install -g vue-cli
$ vue init lianshang-f2e/antd-components my-project
$ cd my-project
$ npm install
$ npm run dev
```

### 构建自己的组件项目

``` bash
vue init lianshang-f2e/antd-components my-project
```
